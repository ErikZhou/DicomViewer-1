# DicomViewer
C# Dicom Test with fo-dicom


Window : RButton Drag (+-1)
  with Ctrl : +-10
  with Ctrl+Shift : +-100
  
ZOOM : Ctrl + LButton Drag

PAN : Shift + LButton Drag
